This folder contains python scripts that use the gravitrax Python Library based on the bleak BLE-Library. 
in order to ise the scripts additional dependencies need to be installed. This can be done by navigating
to the Folder that contains the script and running the following in the commandline:
pip install -r .\requirements.txt

To install all dependencies necessary to use all scripts run the above command with the requirements.txt in this directory. Use
the requirementsRaspPi.txt on a Raspberry Pi to also install the Library necessary to use the GPIO example.

The scripts can be started with the command:
python <scriptname>.py

Examples for the more basic functionallity of the Library can be found in the examples folder in the root 
directory of the Library
